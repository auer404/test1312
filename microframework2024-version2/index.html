Microframework CSS

Ceci est un framework CSS très simple, couvrant certains besoins basiques pour des mises en page HTML. Il propose un ensemble de classes CSS prêtes à être utilisées et permet d'obtenir très rapidement un agencement élégant !

Mise en place

Il suffit de copier la balise suivante dans l'en-tête de la page HTML où vous voulez utiliser Microframework :

<link href="https://auer404.github.io/microframework2024/microframework.css" rel="stylesheet">

Vous aurez alors accès aux fonctionnalités détaillées ci-dessous :

Un système de lignes et de colonnes

Pour mettre en forme une page HTML, la première chose à faire est de déterminer, dans les grandes lignes, comment l'espace s'y subdivise, verticalement et horizontalement. Microframework propose un système permettant de définir des subdivisions de la page dans la hauteur, qui pourront ensuite être elles-mêmes subdivisées dans le sens horizontal.

Ce système fonctionne avec deux classes : ligne et colonne. Une colonne doit se trouver dans une ligne ; une ligne peut contenir plusieurs colonnes. Les différentes colonnes contenues dans une ligne occuperont équitablement l'espace disponible en largeur.

Design responsive et prise en compte des petits écrans

En-dessous d'une largeur de 600 pixels pour la page, les colonnes se placeront toutes l'une sous l'autre (et vous n'avez rien à faire de spécial pour cela).

Exemple :

.colonne
.colonne
.colonne
.colonne
.colonne
.colonne

<div class="ligne">
    <div class="colonne">.colonne</div>
</div>
    
<div class="ligne">
    <div class="colonne">.colonne</div>
    <div class="colonne">.colonne</div>
</div>
    
<div class="ligne">
    <div class="colonne">.colonne</div>
    <div class="colonne">.colonne</div>
    <div class="colonne">.colonne</div>
</div>

Prise en compte des écrans larges

Il est possible d'établir un comportement spécifique de certaines sections au-dessus d'une largeur de 1200 pixels pour la page, en utilisant la classe conteneur1200. En englobant un ensemble d'éléments (par exemples des lignes et colonnes) dans ce type de conteneur, celui-ci sera contraint à une largeur de 1200 pixels maximum, et restera centré horizontalement sur la page.

Exemple :

Ceci est un conteneur1200

.colonne
.colonne

<div class="conteneur1200">
    <p>Ceci est un conteneur1200</p>    
    <div class="ligne">
        <div class="colonne">.colonne</div>
        <div class="colonne">.colonne</div>
    </div>
